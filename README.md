# PointNova Mobile

Prototype web mobile utilisable directement sur Android.

## Ouvrir sur téléphone
1. Décompresse le ZIP.
2. Installe **Acode** depuis le Play Store.
3. Dans Acode, ouvre le dossier `PointNova-Mobile`.
4. Ouvre `index.html`.
5. Appuie sur l'icône d'aperçu / navigateur.

## Mettre sur GitHub depuis Android
Dans le dépôt GitHub vide, appuie sur **uploading an existing file**, puis ajoute :
- index.html
- manifest.json
- sw.js
- icon.svg
- README.md

## Important
C'est une démo locale. Les points, sondages et retraits sont fictifs.
Les vraies pubs, vrais sondages, comptes utilisateurs et paiements nécessitent des services externes et une validation.
